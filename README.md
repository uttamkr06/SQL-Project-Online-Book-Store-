📚 Online Book Store SQL Project

📌 Project Overview

This project is based on an Online Book Store Management System using SQL. It demonstrates how SQL can be used to manage bookstore data, perform data analysis, and generate meaningful insights from multiple related tables.

The project includes three CSV datasets:

Books

Customers

Orders

These tables are connected using common keys to perform various SQL operations such as filtering, joins, aggregation, grouping, and reporting.

🛠️ Technologies Used

SQL (MySQL/PostgreSQL)

CSV Datasets

Database Management System (DBMS)

📂 Dataset

The project uses three datasets:

Books.csv – Contains book details such as title, author, genre, price, stock, and publication year.

Customers.csv – Contains customer information.

Orders.csv – Contains order details including customer, book purchased, quantity, and total amount.

📊 SQL Concepts Covered

SELECT Statements

ORDER BY

GROUP BY

Aggregate Functions

INNER JOIN

HAVING Clause

Subqueries

Sorting & Filtering

Data Analysis Querie

📈 Project Objectives

Retrieve books based on genre and publication year.

Analyze customer purchasing behavior.

Calculate total sales and revenue.

Find best-selling books and top customers.

Track available stock after sales.

Generate business insights using SQL queries.

The project contains:

11 Basic SQL Queries

covering filtering, joins, aggregation, ranking, and analytical reporting.
